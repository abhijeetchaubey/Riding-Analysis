# Riding-Analysis

This project analyzes ride booking data to uncover patterns in customer behavior, ride trends, revenue generation, and operational performance.

The goal is to transform raw ride data into actionable insights using Power BI dashboards that help improve decision-making for ride-sharing platforms.

# Objectives
Analyze ride demand patterns over time
Identify peak booking hours and high-demand locations
Evaluate revenue trends and ride performance
Monitor cancellation rates and booking status distribution
Understand customer and driver behavior patterns
